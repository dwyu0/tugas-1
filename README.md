## 1. Penjelasan Konsep Array
Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data dengan tipe yang sama dalam satu variabel. Pada program ini, array digunakan untuk menyimpan 10 nilai mahasiswa yang diinput oleh user.

## 5. Analisis kompleksitas
Analisis kompleksitas pada program ini digunakan untuk mengetahui efisiensi waktu dan memori dalam pengolahan data nilai mahasiswa. Proses input data dilakukan sebanyak n kali sesuai jumlah nilai yang dimasukkan, sehingga memiliki kompleksitas waktu O(n) dan ruang O(n) karena data disimpan dalam array. Selanjutnya, pencarian nilai tertinggi dan terendah dilakukan dengan memeriksa seluruh elemen dalam array, sehingga memiliki kompleksitas waktu O(n) dan ruang O(1). Perhitungan rata-rata juga dilakukan dengan menjumlahkan seluruh data dalam array, sehingga kompleksitas waktunya O(n) dan ruang O(1). Proses perhitungan jumlah mahasiswa lulus (≥ 60) dilakukan dengan iterasi terhadap seluruh data, sehingga memiliki kompleksitas waktu O(n) dan ruang O(1). Sedangkan pada pembuatan grafik, data yang digunakan merupakan hasil dari proses sebelumnya sehingga hanya membutuhkan kompleksitas waktu O(1) dan ruang O(1). Secara keseluruhan, program ini memiliki kompleksitas waktu dominan O(n) karena sebagian besar operasi memerlukan iterasi terhadap seluruh data dalam array.
