## 1. Penjelasan Konsep Array
Array adalah struktur data yang digunakan untuk menyimpan sekumpulan data dengan tipe yang sama dalam satu variabel. Pada program ini, array digunakan untuk menyimpan 10 nilai mahasiswa yang diinput oleh user.
